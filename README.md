## ğŸ Getting Started

### Prerequisites

- **Node.js** (v18 or higher) - [Install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)
- **Bun** (recommended) or npm
- **Supabase Account** - For backend services

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/humanize-ai.git
   cd humanize-ai
   ```

2. **Install dependencies**
   ```bash
   # Using Bun (recommended)
   bun install

   # Or using npm
   npm install
   ```

3. **Set up environment variables**
   
   Create a `.env.local` file in the root directory:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Start the development server**
   ```bash
   # Using Bun
   bun dev

   # Or using npm
   npm run dev
   ```

5. **Open your browser**
   
   Navigate to `http://localhost:5173` to see the application.

## ğŸ—„ï¸ Database Setup

This project uses Supabase as the backend. The database migrations are located in `supabase/migrations/`. To set up the database:

1. Create a new Supabase project
2. Link your local project: `supabase link --project-ref your-project-ref`
3. Run migrations: `supabase db push`
4. Deploy edge functions: `supabase functions deploy`
